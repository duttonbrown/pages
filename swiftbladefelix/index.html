How to Update Your Site with ChatGPT

You don't need to know code. ChatGPT can make any change to this site for you — just tell it what you want. Here's exactly how.

01

Open your site file

Find the file index.html on your computer (in the swiftbladefelix-site folder). Right-click → Open With → Notepad. Select all (Ctrl+A), copy (Ctrl+C).

02

Go to ChatGPT

Open chat.openai.com — free account works great. Start a new chat.

03

Use this prompt

I have a personal salon website as a single HTML file. I want to [describe your change here]. Please give me the complete updated index.html file with that change made. Keep everything else exactly the same. Here is the current file:

[paste your index.html here]

ChatGPT gives you the updated file. Copy it, paste back into Notepad, save (Ctrl+S).

04

Go live instantly

Drag your swiftbladefelix-site folder onto netlify.com/drop. Your site is live in seconds. Free, no account needed.

Ready-to-use prompts — just fill in the blanks

  • Change hours
    "Update my business hours to: Thursday–Sunday [time]. Monday–Wednesday Closed. Keep everything else the same."
  • Add a service
    "Add a new service called '[name]' to the services list. Give it the next roman numeral. Keep everything else the same."
  • Remove a service
    "Remove '[service name]' from the services list and renumber the remaining services. Keep everything else the same."
  • Update booking link
    "Change all booking links from 'swiftbladehair.com/s/appointments' to '[your new link]'. Keep everything else the same."
  • Update address
    "Update my address to '[new address]' everywhere it appears, including the Google Maps link. Keep everything else the same."
  • Change phone number
    "Update my phone number to [new number] everywhere it appears. Keep everything else the same."
  • Add a review quote
    "Add a customer review quote '[review text]' in the about section below the current one. Keep everything else the same."
  • Swap the photo
    "Replace the salon photo (felix-salon.jpg) with a new image file called '[your-photo.jpg]'. Keep everything else the same." (Put the new photo in the same folder as index.html first.)
  • Change the tagline
    "Change the tagline 'Sharp cuts. Real vibes. Minneapolis made.' to '[your new tagline]'. Keep everything else the same."
  • Add a new section
    "Add a new section after [Services / About / Find Us] called '[section name]' with this content: [describe the content]. Match the style of the other sections. Keep everything else the same."

Make ChatGPT work better for you

  • Always say "keep everything else exactly the same" — stops ChatGPT from redesigning things you didn't ask it to touch.
  • If it only gives you part of the file, say "Give me the complete file, not just the changed section."
  • If something breaks, say "That broke something. Please revert to the original and only make the one change I asked for."
  • Save a backup of index.html before every change (rename it to index-backup.html) so you can always go back.
  • For bigger redesigns, paste the live site URL and say "Here's what it looks like: [URL]. I want to change the color / font / layout of [X]..."
  • Claude (claude.ai) works too and is often better for longer files — same process, same prompts.
